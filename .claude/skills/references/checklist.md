---

## 🔴 CRITICAL — 항상 확인

### 앱 기동
- [ ] `https://mttcs.vercel.app/` 접속 시 로그인 화면 또는 대시보드 정상 표시
- [ ] 브라우저 콘솔에 TypeError / ReferenceError 없음
- [ ] Network 탭에서 500 에러 없음

### 전체 탭 네비게이션
- [ ] 대시보드 → 마스터 필터 이동 정상
- [ ] 마스터 필터 → 스캐너 이동 정상
- [ ] 스캐너 → 신규 계획 이동 정상
- [ ] 신규 계획 → 관심종목 이동 정상
- [ ] 관심종목 → 리스크 이동 정상
- [ ] 리스크 → 뷰티 콘테스트 이동 정상
- [ ] 뷰티 콘테스트 → 매매 복기 이동 정상
- [ ] 매매 복기 → 매크로 이동 정상
- [ ] 매크로 → 가이드/알고리즘 이동 정상
